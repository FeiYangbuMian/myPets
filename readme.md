## 萌宠战记 
> 此项目为毕业设计。后端用Node.js Express框架，前端采用Bootstrap与jQuery。前后端分离，Ajax进行数据交换。

### 已用技术：
> 简单入门node.js后，边学node.js边写，项目难免有点稚嫩0.0
- Express框架
- ejs模板（用不顺手
- Mustache模板（实习单位用的
- mysql一直在用
- formidable处理表单图片
- nodemailer用于邮箱验证
- express-session存储当前回话用户信息
- bootstrap
- jquery
